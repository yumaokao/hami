HamiBook-Scroll
===============

initial
-------
* how to generate this chrome-extension
  code:: sh
  # https://www.npmjs.com/package/generator-chrome-extension
  $ mkdir hamibook-scroll && cd $_ 
  $ yo chrome-extension
  # BDD, ...

  $ gulp babel

* from a cloned repo
  code:: sh
  # git clone
  $ npm install && bower install

build
-----
* build extension
  code:: sh
  $ gulp build
  # in chrome select dist/ directory
